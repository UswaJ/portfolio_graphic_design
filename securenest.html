Secure Nest Platform

Overview: A residential security solution built with Flutter and the MERN stack, featuring AI surveillance integration.

Technical Architecture

I collaborated with the engineering team to build a scalable architecture. We used Flutter for a cross-platform mobile experience and Node.js/Express for a robust backend API. The system processes real-time alerts from AI surveillance cameras.

Frontend Implementation (Flutter)

class SecurityAlert extends StatelessWidget {
  final String alertType;
  final DateTime timestamp;

  const SecurityAlert({required this.alertType, required this.timestamp});

  @override
  Widget build(BuildContext context) {
    return Container(
      padding: EdgeInsets.all(16),
      decoration: BoxDecoration(
        color: alertType == 'Critical' ? Colors.redAccent : Colors.blueGrey,
        borderRadius: BorderRadius.circular(12),
      ),
      child: Text('Alert: $alertType at ${timestamp.toString()}'),
    );
  }
}
